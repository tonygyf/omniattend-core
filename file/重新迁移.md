### `app.py`

```python
from flask import Flask, request, jsonify
import onnxruntime as ort
import numpy as np
from PIL import Image
import base64
import io
import os

app = Flask(__name__)

session = None

def get_session():
    global session
    if session is None:
        model_path = os.path.join(os.path.dirname(__file__), "models/mobilefacenet.onnx")
        session = ort.InferenceSession(model_path, providers=["CPUExecutionProvider"])
        print("Model loaded")
    return session

@app.route("/health", methods=["GET"])
def health():
    return jsonify({"ok": True})

@app.route("/embed", methods=["POST"])
def embed():
    try:
        data = request.get_json()
        image_b64 = data["image"]  # base64 字符串

        # 解码图片
        img_bytes = base64.b64decode(image_b64)
        img = Image.open(io.BytesIO(img_bytes)).convert("RGB")

        # 预处理：112×112，归一化到 [-1, 1]
        img = img.resize((112, 112))
        arr = np.array(img, dtype=np.float32)
        arr = (arr / 127.5) - 1.0
        arr = arr[np.newaxis, ...]  # shape: [1, 112, 112, 3]

        # 推理
        sess = get_session()
        results = sess.run(None, {"img_inputs:0": arr})
        embedding = results[0][0].tolist()  # 128维

        return jsonify({"embedding": embedding})

    except Exception as e:
        return jsonify({"error": str(e)}), 500

if __name__ == "__main__":
    app.run(host="0.0.0.0", port=7860)
```

### 部署步骤（全程免费，不用绑卡）

**第一步：注册 Hugging Face**

打开 [huggingface.co](https://huggingface.co) → Sign Up，用邮箱注册，不需要绑卡。

**第二步：新建 Space**

登录后点右上角头像 → New Space：
- Space name：随便起，比如 `mobilefacenet-server`
- License：MIT
- **SDK 选 Docker** ← 关键
- Visibility：**Private**（不想让别人访问就选这个）
- 点 Create Space

**第三步：上传文件**

进入 Space 后点 Files → Add file → Upload files，把这四个文件传上去：
- `app.py`
- `requirements.txt`
- `Dockerfile`
- `models/mobilefacenet.onnx`

上传完后 Space 会自动开始构建，等 2～3 分钟看到 Running 就好了。

**第四步：拿到你的服务地址**

Space 地址格式是：
```
https://你的用户名-mobilefacenet-server.hf.space
```

测试一下：
```bash
curl https://你的用户名-mobilefacenet-server.hf.space/health
# 返回 {"ok": true} 就成了
```

### Workers 里调用（改一行 URL 就好）

```ts
const INFERENCE_URL = 'https://你的用户名-mobilefacenet-server.hf.space';

async function extractEmbedding(imageUrl: string): Promise<number[]> {
  const imgRes = await fetch(imageUrl);
  const base64 = Buffer.from(await imgRes.arrayBuffer()).toString('base64');

  const res = await fetch(`${INFERENCE_URL}/embed`, {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({ image: base64 })
  });

  const { embedding } = await res.json();
  return embedding;
}
```

### 注意事项

**休眠问题**：免费 Space 48小时无请求会休眠，第一次访问需要等约 30 秒冷启动。如果不想休眠，在 Space 设置里可以升级到付费版，但免费版对考勤系统够用（每天都有人打卡就不会休眠）。

**Private Space 的访问**：如果设成 Private，调用时需要带 Token：

```ts
headers: {
  'Content-Type': 'application/json',
  'Authorization': 'Bearer hf_xxxxxxxx'  // HF Settings 里生成
}
```

**onnx 文件大小**：你的模型是 5.4MB，完全在免费限制内（单文件上限 50MB）。
