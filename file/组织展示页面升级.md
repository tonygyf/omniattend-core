**标题：班级真实人数 + 视图切换 + 考勤任务 500 修复 + AI 常见问题预防**

**摘要**
- 班级管理页：用真实学生人数替换假数据，并支持“班级卡片 / 全部学生按班级分组”切换，记住上次视图。
- AI 洞察页：增加切换按钮，支持“学生列表”和“按班级汇总 + 展开学生”的双视图。
- 考勤任务发布 500：后端不再访问 `CheckinTask/CheckinSubmission`（不存在表），改为复用现有考勤表实现。

**关键改动**
1. **后端（worker）**
   - 修复 500：  
     将 `/api/checkin/tasks*` 与 `/api/checkin/submissions/*` 改为复用 `AttendanceSession / AttendanceResult / PhotoAsset.meta` 实现，不再访问 `CheckinTask/CheckinSubmission` 表。  
     结果：`GET /api/checkin/tasks`、`POST /api/checkin/tasks` 不再 500。
   - `/api/classrooms` 返回真实 `studentCount`：  
     `SELECT c.*, COUNT(s.id) AS studentCount FROM Classroom c LEFT JOIN Student s ON c.id=s.classId GROUP BY c.id`。

2. **数据服务层（services/dataService.ts）**
   - 新增 `fetchClassrooms()`（含 `studentCount`）。
   - 新增 `fetchStudentsByClass()` 给班级分组视图使用。

3. **班级管理页面（pages/Classroom.tsx + pages/Users.tsx）**
   - 移除假数据，改用 `fetchClassrooms()` + `fetchStudentsByClass()`。

     并用 localStorage 记忆上次选择。
